# pi-fs experiment 006: Resonant spiral arm memory and candidate lift

## Goal

Test the next hypothesis:

```text
A resonance does not merely split the index line into residues r.
It creates spiral arms: M = qK + r.
If the arms have memory, they can generate locate candidates better than random.
```

The tested phase orbit is decimal:

```text
u_M = {10^M pi}
```

but phases are quantized into quaternary cells for finite computation.
