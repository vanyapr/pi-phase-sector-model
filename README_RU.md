# Фазово-секторная модель числа pi

English README: [README.md](README.md).

Репозиторий содержит публичный пакет рукописи v7 по фазово-секторной адресации цифровых разложений на примере десятичного разложения числа pi.

Внутри: русская рукопись, экспериментальное приложение, графики, CSV-сводки, распакованные материалы экспериментов 001-018, минимальный демонстрационный код и небольшой Python-модуль точной пропорциональной арифметики секторов.

## Граница утверждений

Основной контур работы:

```text
цель -> пропорция -> конечный фазовый индекс -> сертификат -> адрес
```

В v7 усилена демаркация: это конструктивный декодер и proof-object архитектура, а не статистический предиктор. Известный хвост + алгебраически вычисляемый внешний секторный сертификат + точная пропорциональная арифметика дают сертифицированное декодирование продолжения. Без внешнего сертификата остаётся `10^k` кандидатов для следующих `k` цифр.

Внешний секторный сертификат может быть вычислен из независимого представления числа, например из формулы Мэчина:

```text
pi = 16 atan(1/5) - 4 atan(1/239)
```

Работа не утверждает нормальность pi, не доказывает существование блока из 1000 нулей и не заявляет автономный предиктор новых цифр pi из одного только известного десятичного хвоста. Формализм применим к произвольному вещественному носителю `alpha`; число pi используется как основной экспериментальный пример.

## Структура

- `main_ru_v7.pdf` / `main_ru_v7.tex` - основная русская рукопись.
- `supplement_experiments_ru.pdf` / `supplement_experiments_ru.tex` - экспериментальное приложение 001-018.
- `cover_letter_ru.pdf` / `cover_letter_ru.tex` - русское сопроводительное письмо из submission-пакета.
- `figures/` - корневые графики, использованные в рукописи.
- `tables/` - ключевые CSV-сводки из submission-пакета.
- `code/boundary_continue.py` - извлечение десятичного суффикса из известного хвоста и внешнего секторного сертификата.
- `code/make_certificate.py` - демонстрационный генератор внешнего сертификата через формулу Мэчина.
- `experiments/pi_fs_experiment_001/` ... `experiments/pi_fs_experiment_018/` - распакованные отчеты и артефакты экспериментов.
- `modules/pifs_proportion_module/` - модуль точной пропорциональной арифметики секторов, документация и тесты.
- `docs/roadmap.md` - текущий roadmap репозитория.

Вложенные архивы экспериментов из исходного submission-пакета распакованы в обычные директории, чтобы GitHub показывал реальные файлы.

## Сборка PDF

```bash
lualatex main_ru_v7.tex
lualatex main_ru_v7.tex
lualatex supplement_experiments_ru.tex
lualatex cover_letter_ru.tex
```

TeX-исходники рассчитаны на LuaLaTeX и шрифты `Tinos` / `DejaVu Sans Mono`. Готовые PDF уже лежат в репозитории.

## Тесты модуля

```bash
cd modules/pifs_proportion_module
python3 -m unittest discover
```

Текущая локальная проверка: 6 тестов проходят.

## Демонстрационные скрипты

Скрипт продолжения использует только стандартную библиотеку Python:

```bash
python3 code/boundary_continue.py --help
```

Демо-скрипт сертификата Мэчина дополнительно импортирует `mpmath`. Это намеренно оставлено optional-зависимостью конкретного скрипта, а не runtime-зависимостью всего репозитория.

```bash
python3 code/make_certificate.py --help
```
