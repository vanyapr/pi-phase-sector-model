# Фазово-секторная модель числа pi

English README: [README.md](README.md).

Репозиторий содержит публичный пакет рукописи по фазово-секторной адресации цифровых разложений на примере десятичного разложения числа pi.

Внутри: русская рукопись, экспериментальное приложение, графики, CSV-таблицы, распакованные материалы экспериментов 001-018 и небольшой Python-модуль точной пропорциональной арифметики секторов.

## Граница утверждений

Основной контур работы:

```text
цель -> пропорция -> конечный фазовый индекс -> сертификат -> адрес
```

Положительный результат: конечный сертифицирующий locate-контур и лемма внешнего секторного сертификата. Работа не утверждает нормальность pi, не доказывает существование блока из 1000 нулей и не заявляет автономный предиктор новых цифр pi за публичным вычисленным горизонтом без внешнего сертификата.

## Структура

- `main_ru_v3.pdf` / `main_ru_v3.tex` - основная русская рукопись.
- `supplement_experiments_ru.pdf` / `supplement_experiments_ru.tex` - экспериментальное приложение 001-018.
- `cover_letter_ru.pdf` / `cover_letter_ru.tex` - русское сопроводительное письмо из submission-пакета.
- `figures/` - графики, использованные в рукописи и приложении.
- `tables/` - ключевые CSV-таблицы.
- `experiments/pi_fs_experiment_001/` ... `experiments/pi_fs_experiment_018/` - распакованные отчеты и артефакты экспериментов.
- `modules/pifs_proportion_module/` - модуль точной пропорциональной арифметики секторов, документация и тесты.
- `docs/roadmap.md` - текущий roadmap репозитория.

Вложенные архивы экспериментов из исходного submission-пакета распакованы в обычные директории, чтобы GitHub показывал реальные файлы.

## Сборка PDF

```bash
lualatex main_ru_v3.tex
lualatex main_ru_v3.tex
lualatex supplement_experiments_ru.tex
lualatex cover_letter_ru.tex
```

TeX-исходники рассчитаны на LuaLaTeX и шрифты `Tinos` / `DejaVu Sans Mono`. Готовые PDF уже лежат в репозитории.

## Тесты модуля

```bash
cd modules/pifs_proportion_module
python3 -m unittest discover
```

Текущая локальная проверка: 6 тестов проходят.
